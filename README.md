# AI Movie Recommendation System 🎬

A machine learning-based movie recommendation engine built using Python, Flask, and Vanilla JS.

## 🚀 Features
- **Content-Based Filtering:** Uses Cosine Similarity to find movies similar to your favorites.
- **TMDB API Integration:** Fetches real-time posters for a cinematic UI.
- **Responsive Design:** Works on Desktop and Mobile.

## 🛠️ Tech Stack
- **ML:** Python, Pandas, Scikit-learn
- **Backend:** Flask, Flask-CORS
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript

## 🏃 How to Run
1. Clone the repo: `git clone https://github.com/your-username/your-repo-name.git`
2. Install dependencies: `pip install flask flask-cors requests pandas`
3. Run the backend: `python app.py`
4. Open `index.html` in your browser.
